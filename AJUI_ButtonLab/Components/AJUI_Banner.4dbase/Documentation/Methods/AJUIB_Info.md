<!-- AJUIB_Info ( ) -> version -->



## Description

Show the version info of the component.

```4d
  AJUIB_Info ( ) -> version
```

| Parameter | Type | In/Out | Description |
| --------- | ---- | ------ | ----------- |
| version | text | out | version info of the component |


## Example

```4d
  C_String($version)

  $version:= AJUIB_Info () //AJUI Banner (v18) : 1.7.2-Build21
```
