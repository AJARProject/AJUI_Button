<!-- AJUI_Btn_info ( ) -> version -->


## Description

Show the version info of the component.

```4d
  AJUI_Btn_info ( ) -> version
```

| Parameter | Type | In/Out | Description |
| --------- | ---- | ------ | ----------- |
| version | text | out | version info of the component |


## Example

```4d
  C_String($version)

  $version:= AJUI_Btn_info () //AJUI Button(v18): 1.3.2-Build170
```
